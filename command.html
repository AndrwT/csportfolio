HTML

HTML stands for Hypertext Markup Language and we use it to put content in our websites such as paragraphs and lists.
A HTML paragraph is written by putting certain commands in between <> and ending it with the same command with a

CSS

CSS stands for Cascading Style Sheet and is used to make your website look pretty you give your pages backgrounds and your words a font.
To use CSS you choose a part of the the website you want to change such as the paragraphs, the background, or even a certain font size.

Attributes

Attributes are more specific changes to the website like how many pixels high a word is or the class its in. To change an attribute you type name="value"

Tags

Tags are the things that give the commands specific types of it. For example the p between <> makes a paragraph. A br will make a break in you writing so it won't
go off the page. The ul tag makes an unordered list and the ol tage makes and ordered list.
